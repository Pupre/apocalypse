# 2026-05-04 보건소·급수 초소·지게차 정비고 확장

## 목표

- 야외 대지역이 실제로 서로 다른 생존 질문을 던지도록 의료·관공, 대피선, 물류 벨트에 전용 목적지를 추가한다.
- 단순 전리품 반복을 줄이고, 지역별 정체성에 맞는 큰 결정과 컷신을 연결한다.

## 변경 내용

- `보건소 냉장 약품실`
  - 보냉 가방이 있을 때 온도 유지가 필요한 약품을 선별하는 선택을 추가했다.
  - 보냉 조건이 없을 경우 기본 처치품만 챙기는 현실적인 대안을 둔다.
- `대피선 급수 초소`
  - 물을 모두 가져가는 선택과, 필터를 고치고 공동 몫을 남기는 선택을 분리했다.
  - 물을 든 상태에서 덜 노출되는 우회로 단서를 제공한다.
- `지게차 정비고`
  - 공구와 배터리를 빠르게 챙기는 선택과, 타프·끈으로 임시 운반 보조 장치를 만드는 선택을 분리했다.
  - 물류 구역의 핵심 질문을 “많은 물자를 발견한 뒤 실제로 어떻게 운반할 것인가”로 확장했다.
- 새 컷신 리소스 3장을 `resources/ui/master/indoor/`에 저장하고 UI 리졸버와 매니페스트에 등록했다.

## 검증

- 야외 맵 생성기 재실행 완료.
- 새 이벤트 JSON 파싱, 아이템 참조, 이미지 해상도 확인 완료.
- `res://tests/unit/test_outdoor_map_expansion.gd` 통과.
- `res://tests/unit/test_indoor_director.gd` 통과.
- `res://tests/unit/test_content_library.gd` 통과.
- `res://tests/smoke/test_first_playable_loop.gd` 통과.
- Windows 루트 인증서 저장소 오류와 `content_library_loader_regression`의 의도된 오류 로그는 출력되지만, 테스트 종료 코드는 0으로 통과했다.

## 판단 기록

- 이번 배치는 새 지역을 더 늘리기보다, 이미 깔린 대지역에 플레이어가 기억할 만한 “장소의 질문”을 추가하는 방향으로 진행했다.
- 요구 아이템이 있는 큰 결정은 일부러 유지했다. 장비와 생활 아이템 풀이 커진 만큼, 특정 도구를 들고 특정 장소에 갔을 때 선택지가 열리는 구조가 파밍 기대감을 만든다고 판단했다.
