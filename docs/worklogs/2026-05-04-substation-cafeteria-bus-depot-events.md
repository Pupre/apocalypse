# 2026-05-04 변전소·폐교 급식실·버스 차고 이벤트 확장

## 목표

- 12x12 야외 맵의 대지역이 단순히 넓은 도시처럼 느껴지지 않도록, 발전소 지대·농촌 외곽·국도 구역에 서로 다른 플레이 판단을 추가한다.
- 각 목적지에 전용 일러스트, 실내 구조, 큰 결정 선택지, 위험 피드백, 파밍 보상을 연결한다.

## 변경 내용

- `변전소 제어동`
  - 절연 테이프가 있으면 얼어붙은 차단기를 조작해 송전 마당의 바람길 정보를 얻을 수 있다.
  - 빠르게 배터리팩만 뽑는 선택지도 있어 시간과 소음 사이의 차이를 만든다.
- `폐교 급식실`
  - 쌀 포대를 전부 탐내는 선택과, 실제 가방에 맞게 즉석식·통조림·작은 쌀 봉투를 나눠 담는 선택을 분리했다.
  - 후문과 농로 단서로 농촌 구역 특성을 강화했다.
- `버려진 버스 차고`
  - 노선도를 읽어 검문 우회로를 얻는 선택, 호스와 말통으로 연료를 빼는 선택, 시트 천을 보온 재료로 자르는 선택을 넣었다.
  - 국도 구역의 탐험 보상을 단순 물자보다 이동 경로 정보와 연결했다.
- 세 장소 모두 새 컷신 리소스를 `resources/ui/master/indoor/`에 저장하고 UI 리졸버와 매니페스트에 등록했다.

## 검증

- JSON 파싱과 리소스 해상도 확인 완료.
- 야외 맵 생성기를 다시 실행해 특수 건물 오버라이드가 실제 건물 데이터에 반영되는지 확인 완료.
- `res://tests/unit/test_outdoor_map_expansion.gd` 통과.
- `res://tests/unit/test_indoor_director.gd` 통과.
- `res://tests/unit/test_content_library.gd` 통과.
- `res://tests/smoke/test_first_playable_loop.gd` 통과.
- `content_library_loader_regression`의 의도된 오류 로그는 출력되지만 테스트 종료 코드는 0으로 통과했다.

## 판단 기록

- 이번 배치는 새 맵 전체를 추가로 넓히기보다, 이미 깔린 대지역 구조를 실제 탐색 콘텐츠로 채우는 쪽을 우선했다.
- 사용자가 강조한 “큰 결정 뒤의 몰입감 있는 일러스트”를 반복 가능한 제작 패턴으로 만들기 위해 세 장소 모두 전용 컷신과 테스트 시나리오를 함께 추가했다.
