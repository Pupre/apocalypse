# 2026-05-04 무리한 선택 결과 컷신 패턴

## 목표

- 큰 결정을 성공 보상 컷신만으로 처리하지 않고, 무리한 선택의 결과도 전체화면 일러스트로 보여 준다.
- 플레이어가 “빨리 챙겼지만 시끄러웠다”, “많이 챙겼지만 이동이 망가졌다”는 감정을 즉시 이해하게 만든다.

## 변경 내용

- 새 실패감 컷신 리소스 2장을 추가했다.
  - `indoor_story_noisy_scavenge_failure.png`
  - `indoor_story_overburdened_exit_failure.png`
- 소음형 무리 선택에 소음 결과 컷신을 연결했다.
  - 휴게소 자판기 유리 파손
  - 변전소 배터리팩 강제 회수
  - 버스 차고 시트 절단
  - 지게차 배터리·공구 탈거
- 과적형 무리 선택에 과적 결과 컷신을 연결했다.
  - 폐교 급식실 무거운 쌀 포대
  - 외곽 농가 창고 큰 쌀 포대
  - 대피선 급수 초소 물 전량 회수
- UI 리졸버와 실내 매니페스트에 새 리소스를 등록했다.

## 검증

- 관련 이벤트 JSON 7개 파싱 확인.
- 신규 컷신 이미지 2장 960x480 확인.
- `res://tests/unit/test_outdoor_map_expansion.gd` 통과.
- `res://tests/unit/test_indoor_director.gd` 통과.
- `res://tests/unit/test_ui_kit_resolver.gd` 통과.
- `res://tests/smoke/test_first_playable_loop.gd` 통과.

## 판단 기록

- 이번 배치에서는 확률 실패 시스템을 새로 만들지 않았다. 현재 실내 이벤트 구조가 이미 선택지별 결과 컷신을 안정적으로 지원하므로, 먼저 데이터 패턴을 넓혀 실제 플레이 체감을 올리는 쪽이 더 안전하다고 판단했다.
- 이후 필요하면 이 패턴을 바탕으로 `위험 행동 -> 조건부 성공/실패 컷신` 같은 확률 또는 상태 기반 결과 분기까지 확장할 수 있다.
