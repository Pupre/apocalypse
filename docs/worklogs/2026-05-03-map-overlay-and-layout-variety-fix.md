# 2026-05-03 지도 UI와 지구 내부 반복감 보정

## 배경

- 전체 지도 상단에 범례 텍스트가 좁은 폭으로 밀리면서 한 글자씩 세로로 찢겨 보였다.
- 야외 맵은 지구별 색과 이름은 생겼지만, 같은 지구 안의 블록들이 같은 도로 패턴을 반복해서 "도식화된 도시"처럼 보였다.
- 사용자는 앞으로 자율 배치 중 Git 권한 문제로 멈추지 않기를 원했고, Git 처리는 직접 하기로 했다.

## 구현

- `OutdoorMapOverlay`에서 전체 지도 범례 행을 숨겼다.
  - 현재 정보는 헤더 상태줄과 지도 본문으로 충분히 전달한다.
  - 향후 범례가 필요하면 좁은 상단 행이 아니라 별도 버튼/접힘 패널로 다시 설계하는 쪽이 낫다.
- 야외 맵 확장 생성기에서 `layout_id`를 실제 도로 생성에 연결했다.
  - 북부 시장가: 아케이드형, 광장형, 뒷골목형.
  - 중앙 환승로: 버스 루프형, 역전 교차형, 지하도 우회형.
  - 동부 의료지구: 의료 캠퍼스형, 진료소 진입로형, 약국 마당형.
  - 남부 주거지: 마당형, 연립주택 골목형, 얼어붙은 뒷길형.
  - 남동 공업지대: 적재 야드형, 연료 서비스 부지형, 창고 진입선형.
  - 서부 대피선: 검문형, 구호 동선형, 바리케이드 광장형.
  - 외곽 혼합지대: 서비스 도로형, 눈 덮인 공터형, 끊어진 격자형.
- 지구별 건물 앵커 후보를 늘려 같은 지구 안에서도 건물 배치가 덜 반복되게 했다.
- 생성기를 다시 실행해 외곽 135개 블록 파일을 갱신했다.

## 운영 규칙 변경

- 앞으로 Codex 자동 배치에서는 `git add`, `git commit`, `git push`를 하지 않는다.
- 작업 후에는 변경 파일과 검증 결과만 보고하고, 사용자가 직접 확인한 뒤 Git 처리를 한다.
