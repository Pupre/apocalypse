# 2026-05-03 장비 로드아웃 관리 기능

## 배경

아이템 풀이 커지면서 장비 아이템도 늘었지만, 기존 가방 UX는 착용 상태를 보여주는 데 그쳤다. 플레이어가 “지금 무엇을 입고 있고, 무엇을 바꾸면 어떤 판단이 필요한지”를 빠르게 이해하려면 장비 칸이 단순 표시가 아니라 조작 가능한 로드아웃 영역이어야 한다.

## 구현

- 가방 상단 장비 스트립을 4열 그리드로 바꿨다.
- `등/몸/외투/머리/목/얼굴/손/발/양말/장갑 안감/허리/주머니` 칸을 모두 표시한다.
- 착용 중인 장비 칩에는 아이콘, 장비명, 핵심 효과 한 줄, `해제` 버튼을 보여준다.
- `RunState.unequip_slot()`을 추가해 장비 해제를 실제 상태 변화로 처리한다.
- 실내 인벤토리 액션은 `unequip_inventory_slot_*` 액션을 받아 장비를 가방으로 되돌리고 파생 수치를 다시 계산한다.
- 새 장비를 선택했을 때 같은 슬롯에 이미 장비가 있으면 “무엇과 교체되는지”를 상세 시트에 알려준다.

## 검증

## 다음 확장 후보

- 장비 칸을 눌렀을 때 즉시 해제가 아니라 상세 비교 팝업으로 들어가는 흐름.
- 장비별 내구도와 젖음/오염 상태.
- 착용 조합 보너스와 계절/날씨별 장비 추천.
