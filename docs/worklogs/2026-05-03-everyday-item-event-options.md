# 2026-05-03 생활 아이템 이벤트 선택지 연결

## 목표

- 새로 추가한 생활 아이템과 응용 조합이 도감/가방에만 머물지 않고, 실제 건물 탐색 판단으로 이어지게 한다.
- “이 물건이 왜 필요하지?”가 아니라 “이 상황이면 이 물건을 써볼 수 있겠다”는 느낌을 실내 선택지에서 먼저 보여준다.

## 적용 내용

- 편의점 계산대에 `유리 수색 장갑으로 계산대 안쪽을 끝까지 뒤진다` 선택지를 추가했다.
  - `evd_glass_search_gloves`가 있으면 잠금 해제된다.
  - 기존 급한 계산대 수색의 깨진 유리 체력 손실을 피하고, 립밤/교통카드/복권 같은 생활감 있는 작은 물건까지 찾을 수 있다.
  - 성공 시 `indoor_story_convenience_glass_counter_success.png` 전체화면 컷신을 띄운다. 이 이미지는 생성형 이미지로 만든 뒤 960x480 규격으로 크롭/리사이즈해 `resources/ui/master/indoor/`에 보관했다.
- 카페에 두 가지 생활 도구 선택지를 추가했다.
  - `evd_clean_water_prefilter`로 커피 머신 물통을 걸러 물 보급원을 확인한다. 필터는 커피 찌꺼기로 막혀 소모된다.
  - `evd_window_gap_roll`로 창가 틈바람을 막고 좌석 아래를 더 깊게 뒤진다. 설치한 틈막이는 해당 장소에 남는 것으로 처리해 소모된다.
- 코인 세탁소에 젖은 바닥/세제 위험 대응 선택지를 추가했다.
  - `evd_foot_dry_kit`가 있으면 세탁기 하부를 더 깊게 뒤진다. 키트는 장비형 준비물로 남는다.
  - 일회용 장갑, 한 짝 고무장갑, 유리 수색 장갑 중 하나가 있으면 새는 세제통 뒤쪽을 뒤질 수 있다.
- 낡은 아파트에 생활 응용 선택지를 추가했다.
  - `evd_door_draft_snake`로 101호 문 밑을 막고 방 안쪽을 더 오래 탐색한다.
  - `evd_foot_dry_kit`로 공용 세탁실의 젖은 빨랫감 더미를 더 안정적으로 뒤지며, 보일러실 단서 흐름도 유지한다.

## 검증

- 신규 테스트 `res://tests/unit/test_everyday_item_event_options.gd`를 추가했다.
  - 아이템이 없으면 관련 선택지가 잠긴 상태로 보이는지 확인한다.
  - 아이템을 추가하면 선택지가 잠금 해제되는지 확인한다.
  - 소모형 도구와 장비형 도구의 소모 여부가 의도대로 나뉘는지 확인한다.
  - 선택지 해결 후 장소 플래그, 물 보급원, 단서 공개가 정상 반영되는지 확인한다.
  - 편의점 유리 계산대 선택지가 스토리 컷신 페이로드를 남기는지도 확인한다.
- `res://tests/unit/test_ui_kit_resolver.gd`는 새 컷신 이미지가 UI 리졸버를 통해 960x480 텍스처로 로드되는지 확인한다.
